# **App Name**: NexoForge Landing

## Core Features:

- Headline and Value Proposition: Compelling headline and subheadline to immediately convey NexoForger's value proposition to enterprise businesses.
- Services Section: Detailed sections outlining the specific tech services offered by NexoForger, highlighting their benefits and capabilities.
- Customer Testimonials: Showcase customer testimonials with company logos to build trust and credibility.
- Call-to-Action: Prominent 'Request a Consultation' button with clear instructions.
- Mobile Responsiveness: Ensuring the landing page is fully responsive and optimized for viewing on various devices.
- Scroll-triggered Animation: A Sqwiggle element which is revealed on the page and whose movements follow the scroll actions (parallax animation).

## Style Guidelines:

- Primary color: #FA779B, a vibrant and modern color.
- Secondary color: #A02368, providing depth and sophistication.
- Accent color: #18E5E9, drawing attention to key interactive elements.
- Background color: Very light grey (#F5F7FA), creating a clean and modern look.
- Complementary color: #43B17A, adding balance and visual interest.
- Black: Absolute black for text and key elements to provide contrast and clarity.
- Font: 'Inter', sans-serif. Suitable for headlines and body text, creating a clean and very techie aesthetic.
- Incorporate subtle gridlines, like logo design templates, as an underlay throughout sections, providing a glassy look.
- Design the website with a very modern and techie aesthetic to mirror NexoForger's brand and image.
- Subtle animations on hover and scroll to enhance user engagement.
- Dark And Light theme switcher
